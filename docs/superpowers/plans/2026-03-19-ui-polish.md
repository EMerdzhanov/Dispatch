# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all inline styles with a proper CSS design system, fix layout overflow issues, and produce a polished terminal manager UI.

**Architecture:** Single CSS file (`dispatch.css`) with CSS custom properties for the design token system. BEM-lite class naming (`d-component`, `d-component--modifier`). No Tailwind (unreliable in our Electron webpack build). All components switch from inline `style={}` to `className` references.

**Tech Stack:** Pure CSS with custom properties, existing React components, existing webpack css-loader pipeline
